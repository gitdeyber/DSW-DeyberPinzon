ETIQUETAS EN HTML5

Las etiquetas o tag fueron creadas desde el comienzo en internet, con ellas puede crear una pagina web, su escritura para que el idioma sea entendido se escribe con el signo menor y mayor "< >" y dentro de estos simbolos se escribe el tag html a usar. agunos ejemplos:

!Doctype:

Define el tipo de documento, da a entender a nuestro navegador la version de html que se esta usando.

Article:

Define un articulo, y permite que los contenidos sean tratados independientemente del documento que lo contengan.

Audio:

Define un contenido con audio.

Aside:

Define un contenedor lateral de una pagina.

Body:

Define el cuerpo del documento de la pagina WEB.

ATRIBUTOS EN HTML:

Los atributos son propiedades o valores adicionales que contienen los elementos, estas etiquetas ajustan los comportamientos de los elementos para una ejecucion especifica del usuario algunos ejemplos:

Id:

permite definir un elemento en un documento de forma uniquevoca, NO puede existir dos elementos con el mismo valor de elemento id.

Style:

permite definir el estilo de un elemento como el color.

Title:

esta pripiedad se usa en forma de tip, cuando cambia de color un elemento si pasa el mouse por encima de este.

onclic:

Evento del raton cuando se hace un clic.

ondblclic:

Evento cuando se relaiza un doble clic con el raton.
Referencia: MDN webdocs. 2021. Referencia de atributos HTML. Recuperado de: https://developer.mozilla.org/es/docs/Web/HTML/Attributes

VALORES EN HTML:


Son valores que se asignan las propiededaes de los atributos que contengan el elemento, por ejemplo para definir el lenguaje de una pagina si es ingles (en), español (es) o frances (fr) etc.
los atributos deben de estar unidos por el signo "=" los valores deben estar encerrados por comillas. algunos ejemplos.
