CSS

Historia

CSS surgio de la necesidad de tener una herramienta que aplicara de forma solida para construccionn de las páginas electrónicas, despues de los años 1970 y con el crecimiento del internet la W3C (World Wide Web) organización encargada de la creació de estandares para la WEB, propuso un lenguaje de estilos especifico de programación para los documentos HTML; de las propuestas que se tuvieron en cuenta CHSS (cacascading HTML style sheets) por Hakun Wium Lie y SSP (Stream-basedStyle Sheet Proposal) por Bert Bos, estos dos personajes se unieron para definir un nuevo lenguaje al que lo llamaron CSS (Cascading Style Sheets), W3C hacia el año 1995 publicó la primera recomendación conocida como CSS 1 para 1998 se publico la segunda Version de CSS, esta es la mas usada por por los programadores, para el año 2009 lanzan la version 2003 pero esta desde el 2008 aun sigue en mejoras.
.

CSS es un lenguaje de programacion encargada de darle estilo a las hojas de electrónicas programadas en la internet, esta herramienta es la mejor forma de separar los contenidos de la presentación, esto ayuda para la creacion de paginas WEB complejas, esto obliga a que las paginas sean creadas con dos tipos de herramientas para Páginas WEB, unaes HTML qu en este moneto esta en su version 5, y CSS que esta en version 3, la ventaja de programar con estas dos herramientas radica en la facilidad para su mantenimiento, y permite que la pagina pueda ser visualizada por diferentes equipos. una ves creados los contenidos con HTML, entra a trabajar CSS para definir el aspecto de cada elemento de la página, su color, tamaño de la letra, fuente de la letra, posicion etc.

Algunos comandos:


Absoluta:


Establecen un valor fijo, este comando no permite la modificación del elemento.
Relativa: son medidasflexibles, dejan manipular los elementos deacuerdo al diseño del porgramador.
px: unidades de pixeles permite programar el tamaño, son las mas usadas por los programadores.
mm: milimetros
in: pulgadas.
%: porcentaje.
em: toma el tamaño de la fuente mas cercano.

text-align:

alinea alineacion de texto.

Color:

Define el color del texto.

backgroung-color:

define el color del fondo de pantalla.

Background-image:

imagen de fondo.

Background-repeat:

Repetir la imagen de fondo.

Font-Style

estilo de letra.

:hover

Modifica el estilo cuando un apuntador pasa por encima de un elemento.

Display:

comportamiento del contenedor.
